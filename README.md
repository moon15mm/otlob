# اطلب (Otlob)

> منصّة SaaS متعددة المستأجرين تحوّل المطاعم إلى بوت طلبات آلي على واتساب — نسخة المطاعم من نظام «وردات».

## 📋 الوصف

«اطلب» يتيح لكل مطعم استقبال طلبات الزبائن عبر واتساب بالذكاء الاصطناعي: عرض قائمة الطعام (المنيو)،
بناء **سلة طلب متعددة الأصناف** بكميات، اختيار التوصيل أو الاستلام، والدفع (إلكتروني/نقدي/تحويل بنكي).
كل مطعم له بيانات اعتماد واتساب وذكاء اصطناعي وبوابة دفع خاصة به، ولوحة تحكم مستقلة.

## 🛠️ التقنيات المستخدمة

- TypeScript + Node.js + Express
- Prisma + PostgreSQL
- OpenAI / Gemini
- Baileys / WhatsApp Cloud API
- Stripe / Moyasar / Tap / MyFatoorah

## 🚀 تعليمات التشغيل

```
npm install
npx prisma generate && npx prisma db push
npm run seed       # مطعم ديمو بأصناف نموذجية
npm run build && npm start
# أو للتطوير:
npm run dev
```
