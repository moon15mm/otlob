# Project Info: otlob (اطلب)

## 📄 الوصف العام
Otlob — WhatsApp AI Restaurant Ordering Automation (نسخة المطاعم من نظام وردات)

## 🛠️ التقنيات المستخدمة
- Node.js + TypeScript + Express + Prisma + PostgreSQL

## 🚀 تعليمات التشغيل الدقيقة
1. npm install
2. npx prisma generate && npx prisma db push
3. npm run seed
4. npm run build
5. npm run start  (أو npm run dev للتطوير)
